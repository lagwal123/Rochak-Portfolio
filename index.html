my skills

HTML

The backbone of web pages, HTML structures content using elements like headings, paragraphs, and images. It provides the semantic foundation for accessibility and SEO(Search Engine Optimization).

CSS

CSS styles the HTML structure, enhancing design with colors, layouts, and animations. It enables responsive and visually appealing web experiences.

RWD

RWD ensures websites adapt to different screen sizes using flexible grids, media queries, and fluid layouts. It improves usability across desktops, tablets, and mobile devices.

Bootstrap

A powerful CSS framework that simplifies web development with pre-designed components, grids, and responsive utilities. It helps create modern, mobile-friendly websites quickly and efficiently.

Wordpress

WordPress is a free tool that helps you easily create and manage your own website or blog without needing to write a lot of code.

Woo commerce

WooCommerce is a free tool (plugin) that you add to WordPress to turn your website into an online store where you can sell products, take payments, manage orders, and run your own shop online.

Shopify

Shopify is an online platform that lets you build, manage, and run your own online store to sell products or services on the internet or in person. It gives you tools for things like product pages, payments, shipping, inventory, and marketing — all in one place.

GSAP

GSAP (short for GreenSock Animation Platform) is a JavaScript library developers use to make smooth, powerful animations on websites — like moving things, fading elements in and out, or creating fancy scrolling effects.

JavaScript

JavaScript is a language that tells the computer how a webpage should behave and do things, not just look like static text.It works together with HTML (which gives structure to the page) and CSS (which makes it look nice).

React

React.js (often just called React) is a popular open-source JavaScript library for building user interfaces, especially for modern interactive web applications like single-page apps (SPAs).
